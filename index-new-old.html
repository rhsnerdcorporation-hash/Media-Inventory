Connect to Google Sheets

The app talks to your spreadsheet through a small Google Apps Script web app. It keeps an Inventory sheet in sync with every item's status, a Kits sheet with your saved equipment bundles, and an Activity Log sheet recording every check-out, check-in, and missing report. Follow these steps to set it up — it takes about 5 minutes.

  1. 1

    Create a Google Sheet

    Go to sheets.google.com and create a new, blank spreadsheet. Name it "Media Equipment Log". You don't need to add any headers or columns yourself — the script creates the Inventory, Kits, and Activity Log tabs automatically the first time it runs.

  2. 2

    Open Apps Script

    In your sheet, click Extensions → Apps Script. Delete any existing code in Code.gs and paste in the script below (this is the code.gs file from the app download — open it and copy everything).

  3. 3

    Paste this script

    function doGet(e) {
  var action = (e.parameter && e.parameter.action) || 'inventory';
  if (action === 'inventory') {
    return jsonOut({ items: readInventory() });
  }
  return jsonOut({ error: 'Unknown action: ' + action });
}

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    if (data.action === 'sync_item') {
      upsertInventoryRow(data.item);
      return jsonOut({ ok: true });
    }
    if (data.action === 'bootstrap_inventory') {
      bootstrapInventory(data.items || []);
      return jsonOut({ ok: true });
    }
    appendLogRow(data); // out / in / missing
    return jsonOut({ ok: true });
  } catch(err) {
    return jsonOut({ error: err.message });
  }
}

/* …plus the helper functions for reading and
   writing the Inventory and Activity Log sheets.
   See the full code.gs file for the complete script — 
   it's too long to show here in full, this is just a
   preview of the two entry points. */

    This is just a preview of the two entry points — paste the entire contents of the code.gs file you downloaded with the app, not just what's shown above.

  4. 4

    Deploy as a web app

    Click Deploy → New deployment. Set type to Web app. Set "Execute as" to Me and "Who has access" to Anyone. Click Deploy and authorise when prompted.

  5. 5

    Paste the URL into the HTML file

    Copy the web app URL (looks like https://script.google.com/macros/s/…/exec). Open the app's HTML file in a text editor, find the SHEETS_API_URL constant near the top of the <script> section, and paste the URL between the quotes. Save the file and reload the page — the app will immediately try to load your inventory from the sheet, and since it's empty the first time, it instead pushes your current local inventory up to seed it. The status dot in the sidebar turns green once that's done.
